# 프로젝트 소개
- 특정 API 를 통해 목록을 테이블화 시킨 페이지와, canvas 를 통한 이벤트가 존재하는 상세 화면 페이지로 구성된 홈페이지 입니다.

# 개발 환경
- CRA(create-react-app)
- OS : MAC
- node version : v18.6.0
- npm version : v8.13.2

# 실행 방법
- npm install
- npm run start
